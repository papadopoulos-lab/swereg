# Delete all skeleton output files from disk

Removes all \`skeleton_BBB_SS.qs2\` files from the skeleton directory.

## Usage

``` r
skeleton_delete_skeletons(skeleton_meta, ...)
```

## Arguments

- skeleton_meta:

  A \[SkeletonMeta\] object.

- ...:

  Not used.

## Value

The \[SkeletonMeta\] with empty \`skeleton_files\`.
