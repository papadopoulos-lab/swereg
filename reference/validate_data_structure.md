# Validate data structure for add\_\* functions

Validate data structure for add\_\* functions

## Usage

``` r
validate_data_structure(data, required_cols = NULL, data_type = "data")
```

## Arguments

- data:

  A data.table containing the data

- required_cols:

  Character vector of required column names

- data_type:

  Character string describing the data type (for error messages)

## Value

Nothing if valid, stops with error if invalid
