# Load a TTE plan from disk

Reads a \`.qs2\` file saved by \`\$save()\` and reconstructs a
\[TTEPlan\] object.

## Usage

``` r
tte_plan_load(path)
```

## Arguments

- path:

  File path to the \`.qs2\` plan file.

## Value

A \[TTEPlan\] object.
