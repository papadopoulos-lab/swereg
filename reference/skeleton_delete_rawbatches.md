# Delete all rawbatch files from disk

Removes all \`BBB_rawbatch_group.qs\` files from the rawbatch directory.

## Usage

``` r
skeleton_delete_rawbatches(skeleton_meta, ...)
```

## Arguments

- skeleton_meta:

  A \[SkeletonMeta\] object.

- ...:

  Not used.

## Value

The \[SkeletonMeta\] with empty \`groups_saved\`.
