# Validate pattern list for diagnoses/operations/prescriptions/cods

Validate pattern list for diagnoses/operations/prescriptions/cods

## Usage

``` r
validate_pattern_list(pattern_list, pattern_type = "patterns")
```

## Arguments

- pattern_list:

  Named list of patterns

- pattern_type:

  Character string describing the pattern type (for error messages)

## Value

Nothing if valid, stops with error if invalid
